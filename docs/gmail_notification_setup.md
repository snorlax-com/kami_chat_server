# AuraFace 相談メール通知 — Gmail での見分け・フィルタ・通知設定

開発者向けに、**通常相談**と**優先導き**の通知を Gmail 上で分け、見逃しにくくするための手順です。

## メールの違い（一覧）

| 項目 | 通常相談 | 優先導き |
|------|----------|----------|
| `consultationType` | `normal` | `priority_guidance` |
| 件名（例） | `[AuraFace] 新しい相談が届きました` | `【要確認】【優先導き】2時間以内対応｜AuraFace [PRIORITY_GUIDANCE]` |
| 本文の目印 | 「通常相談」 | 冒頭に「【優先導き】」「2時間以内対応」、識別子 `[PRIORITY_GUIDANCE]` |

## 1. Gmail でフィルタを作成する

1. Gmail を開き、上部の**検索ボックス**右の**フィルタアイコン**（スライダー）をクリックします。
2. **件名**欄に次のいずれかを入力します。
   - 優先導きのみ拾う例: `優先導き`
   - より絞る例: `【要確認】【優先導き】`
3. **検索**を押して件数を確認し、問題なければ**フィルタを作成**をクリックします。

### 推奨の検索クエリ例（コピー用）

- `subject:優先導き`
- `"【要確認】【優先導き】"`
- `[PRIORITY_GUIDANCE]`

## 2. フィルタの動作（推奨）

フィルタ作成画面で、次を組み合わせると運用しやすくなります。

- **ラベルを付ける**: 新規ラベル「優先導き」を作成して付与
- **重要マークを付ける**: オン（利用できる場合）
- **迷惑メールにしない**: オン
- 必要に応じて**スターを付ける**: オン

通常相談用に別フィルタを作る場合は、件名に `新しい相談が届きました` を含む、などと指定できます。

## 3. 通常相談と優先導きの見分け方

- **通常相談の件名**: `[AuraFace] 新しい相談が届きました`
- **優先導きの件名**: `【要確認】【優先導き】2時間以内対応｜AuraFace`（末尾に `[PRIORITY_GUIDANCE]` が付く場合があります）

ロック画面の通知でも、件名に「優先導き」「要確認」が入るため優先度が分かりやすくなります。

## 4. Android 版 Gmail アプリで通知を強くする（基本）

端末や Gmail のバージョンでメニュー名が多少異なる場合があります。

1. **Gmail** アプリを開く
2. 左上の**メニュー**（三本線）を開く
3. **設定**をタップ
4. 通知を受け取りたい**アカウント**を選択
5. **通知**を開き、メールの受信通知が**すべて**または**高優先度のみ**など、運用に合う設定になっているか確認
6. （利用できる場合）**ラベルの通知**や**ラベルごとの設定**で、「優先導き」ラベルに**通知をオン**にする
7. 端末の**設定 → アプリ → Gmail → 通知**でも、チャネルごとに音や表示を調整できる機種があります

## 5. iPhone 版 Gmail で確認すべき点

1. **設定（iOS）→ 通知 → Gmail** で通知が**許可**されているか
2. **Gmail アプリ**を開き、**設定 → [アカウント] → 通知**で、アプリ内の通知が有効か確認
3. **重要メール**や**優先表示**に関する項目があれば、開発者用受信箱の運用に合わせてオン

※ iOS はシステム側の通知設定が優先されるため、Gmail 単体で「このラベルだけ別音」とまではできない場合があります。その場合はフィルタ＋ラベル＋バッジ表示の組み合わせで運用します。

## 6. Gmail 検索用キーワード

- `subject:優先導き`
- `"【要確認】【優先導き】"`
- `[PRIORITY_GUIDANCE]`
- `subject:"新しい相談が届きました"`（通常相談のみ）

## 7. 見逃し防止の運用

- 優先導きのメールは、**返信・対応が終わるまでアーカイブしない**習慣にする
- **「優先導き」ラベル**を受信後すぐ確認する（フィルタで自動付与）
- アプリ側の**管理者返信ページ**や未対応一覧と併用する
